# GrabGrocery 

GrabGrocery is a grocery shopping website inspired by platforms like Blinkit, Zepto, Instamart, and BigBasket.

## Project Status

Currently under development.

## Features

- Home Page
- Product Listing
- Search Bar
- Shopping Cart
- Login & Registration
- Responsive Design
- JavaScript Functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript


## Goal

The goal of this project is to build a modern grocery shopping website while learning front-end web development from scratch.
